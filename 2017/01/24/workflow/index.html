Workflow

购买了 Workflow 之后，你果的设备生产力提高了两台 Nexus

正如 官方 所宣称的

Spend less taps, get more done.

这里将列举一些有趣的 Workflow（虽然官方已经加入了 gallery 功能

那就打造成个人向的收藏夹吧

Day One Coffee Log

Create a Day One entry for coffee tasting notes.

及其便利的在 Day One 中记录咖啡记录，便利的记录产地，烘焙度，口感还有设备等信息

Pocket to Instapaper

RT, 自从 Instapaper 免费了之后，有考虑从 Pocket 迁移到 Instapaper 去，不过也仅仅只是考虑了一下下，还是 Pocket 好用

网易云音乐 ➡️ Apple Music

同样也是迁移用，Apple Music 有教育优惠很棒棒哦

Send To Kindle

Send the current web page to your Kindle.

比起复制链接到微信里打开再分享到亚马逊的服务号还是高效很多的

万能扫码

现支持支付宝、淘宝、京东、微信、微博、豆瓣、大众点评、115。* 需跳转到 App 本身二次扫码，仍在完善中…

在 sspai 上发现的，终于不用每次问老板这是啥的二维码然后又去找软件了

快递查询

同样来源于sspai，作者提供了几种查快递的方式（大概在快递 100 不修改 api 之前都能使用吧
